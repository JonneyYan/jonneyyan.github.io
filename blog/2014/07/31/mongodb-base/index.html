MongoDB Base

基本概念：

  • 文档（Document） 是MongoDB中的基本数据单元，与关系型数据库中的记录（行）类似，但更富表现力
  • 集合（Collection） 可以看做拥有动态模式的表
  • 每个文档都有一个特殊的键_id，这个键是文档的唯一标识
  • MongoDB自带一个JavaScript Shell，用于管理MongoDB
